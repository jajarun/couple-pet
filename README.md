# 分身宠物 · 情侣版

> 你养的宠物,是 AI 扮演的「另一半」。

一个给情侣两个人玩的手机网页小游戏:你们各自捏一只**代表对方**的分身宠物,用它来隔空互动——骂两句、喂狗粮、抱一下、说想你、闹别扭又和好。分身由 AI 驱动,始终在人设里、无厘头、绝不承认自己是 AI;真人也能随时"附身"顶替它说话。核心是把两个人日常那点闹腾、想念、撒糖,变成一只会回话的宠物。

---

## 这是什么

- **一对一配对**:一方创建情侣、拿到邀请码,另一方输码加入,绑成一对。
- **两只镜像分身**:配对后自动生成两只分身——你养的那只代表 TA,TA 养的那只代表你。你在你的分身身上做的动作,会出现在 TA 的时间线里。
- **共享的关系数值**:委屈、狗粮、想你、亲密四个值两人共用,随互动和时间变化,是这段关系此刻的"温度计"。
- **AI 分身 + 真人附身**:平时分身用 AI 按人设回你;真人也能亲自下场,用"本尊回应"顶替 AI 说这句话。

---

## 核心功能

### 🎭 捏分身
给你眼里的 TA 设定一只宠物:**基调**(沙雕 / 傲娇 / 温柔…)、**名字**、**emoji 形象**、一段**种子人设**(TA 是个什么样的人),以及**性别**。这些会揉进 AI 的提示词,决定它怎么跟你说话。

### 💢 九种互动动作
每个动作既改数值、也会引出一句回应:

| 动作 | 数值变化 | 回应来源 |
|---|---|---|
| 骂一顿 `scold` | 委屈 +15 | 🤖 AI 回怼 |
| 戳一下 `poke` | 委屈 +5 | 本地沙雕文案 |
| 喂狗粮 `feed_dogfood` | 狗粮 +20,委屈 −10 | 本地沙雕文案 |
| 抱抱 `hug` | 想你 −30,亲密 +10 | 本地沙雕文案 |
| 说想你 `miss_you` | 想你 −20,亲密 +6 | 本地沙雕文案 |
| 道歉 `apologize` | 委屈 −25,亲密 +8 | 本地沙雕文案 |
| 唠两句 `chat` | — | 🤖 AI 按人设回 |
| 哄回来 `coax` | 委屈 −30,亲密 +5 | 本地沙雕文案 |
| 摸摸头 `headpat` | 亲密 +3 | 本地沙雕文案 |

> 便宜动作走**本地文案**(每个动作 50+ 条,尽量不重样,不烧 API);**骂 / 唠嗑**才叫 AI,省钱又不掉品质。
>
> 两个动作是**条件解锁**的、平时不在动作区里:`coax`(哄)只在分身**离家出走**时出现,`headpat`(摸摸头)只在**两人同框**时出现。后端也各自拦一道,防多端绕过。

### 📊 四个关系数值(会自己变)
- **委屈**、**狗粮**、**想你**、**亲密**,范围 0–100。
- **随时间漂移**:想你每小时 +4(越久越想),委屈每小时 −2、狗粮每小时 −1(慢慢平复/消化);**亲密不随时间变**,只靠互动积累。
- **委屈 ≥ 80** 触发系统旁白「该哄了」,提醒你去哄。

### 🥚 分身进化(你怎么养,它就长成什么样)
分身会长大:**蛋 🥚 → 幼体 🐣 → 成体 → 完全体**。推动它的**不是**共享数值,而是「**这只分身的饲养者对它做过什么**」——每个动作按权重攒经验(抱/喂/哄 3 分,聊/想你/道歉 2 分,戳/骂 1 分)。

进成体那一刻按动作占比**定型分支,此后不可逆**:

| 分支 | 怎么养出来的 | 成体 → 完全体 |
|---|---|---|
| 甜妹 `sweet` | 抱 / 想你 / 道歉 / 哄 占比最高 | 🐰 甜妹雏形 → 🦄 甜心精灵 |
| 猪猪 `glutton` | 一个劲喂狗粮 | 🐷 小猪猪 → 🐽 猪猪本猪 |
| 黑化 `dark` | 骂 / 戳 占比最高 | 😼 腹黑体 → 😈 黑化完全体 |
| 话痨 `chatty` | 光顾着唠嗑 | 🐸 话痨体 → 🦜 复读机精 |
| 均衡 `balanced` | 什么都来一点(或平局) | 🐱 均衡体 → 🐲 六边形战士 |

> 没有哪个桶占到 45% 就是均衡。蛋 / 幼体阶段还用你捏分身时选的 emoji,**成体起形态才抢过来**(不会覆盖你存的 `appearance.emoji`)。

两只镜像分身**各长各的**,于是:

- 🏠**TA 页签**看到的是**你养的那只**(代表 TA)——你把 TA 养成了什么样;
- ⚙️**我页签**看到的是**TA 养的那只**(代表你)——**你在 TA 眼里被养成了什么样**。

被养成黑化毒舌兽,还是被喂成一只猪,一眼就知道。

### 🤖 AI 回应(DeepSeek 驱动)
分身回话时,提示词会结合:**人设 + 基调 + 性别 + 进化形态 + 此刻心情**(某个值 ≥60 就带上对应情绪,比如委屈高就"带刺想被哄")**+ 最近若干条对话上下文**。风格固定为无厘头、1–3 句、口语、绝不承认自己是 AI。
- **形态会反过来改写它怎么说话**:黑化版毒舌带刺、猪猪版满脑子吃的、甜妹版黏人撒娇。你怎么对它,它就怎么回你——玩法和 AI 咬合成闭环。
- **API Key 只在服务端**,前端永远看不到。
- **没配 Key 也能玩**:自动走本地沙雕兜底,骂/聊天也有话说(适合离线 / 自测)。
- 每人每天有 AI 次数上限(默认 50 次),防跑量。

### 😽 分身主动撩你
页面开着没人理时,分身会**自己冒泡**来撩你/找话题——结合当前的委屈、狗粮、想你、亲密和 TA 的性别人设生成。空闲一小会儿(默认 55 秒)才触发,不刷屏。有 Key 走 AI,没有就用本地"撩人"文案兜底。

### 🌙 分身梦话
每天早上 8 点(UTC+8),分身会冒出一句**关于昨夜的梦话**,结合它的形态和此刻心情生成——黑化的连做梦都在阴阳怪气,猪猪版梦里还在吃。首页一张小卡,推送一条提醒。**每天早上都有个新东西等你**,这是"时不时点进来看一眼"的钩子。

> 为什么不在凌晨生成:Web Push 会在手机上响,绝不能半夜推。早晨生成、内容讲昨夜,你任何时候打开都看得到。

### 🪹 分身离家出走
**你在 1 小时里骂满 5 次,一句好话都没有**——第 5 次骂发出去的那一瞬间,它留张纸条就跑了(没有定时任务,当场生效)。这一下它连嘴都懒得回。首页变成一个空窝,除了「🥺 去哄它回来」什么都点不了(后端也拦,防多端绕过)。

**但哄只能把它哄到门口。** 回不回去,得**它代表的那个人**说了算——那只分身养在你家,可它是「TA」啊。于是:

| 谁 | 看到什么 | 能做什么 |
|---|---|---|
| 骂跑它的人(keeper) | 空窝 + 纸条 | 「🥺 去哄它回来」→ 哄完变成「🙏 等 TA 点头…」,连哄都点不动了 |
| 它代表的那个人 | 首页多一张卡:「TA 把「你」气跑了」 | TA 来哄了才出现「💌 原谅 TA」;没来哄就晾着 |

判据**只看你自己的行为**,不看共享数值,所以:

- **只有「骂」算数**——`poke` 是点一下分身就会发的动作,手滑五下不该把它逼走;
- 两只镜像分身**各跑各的**——谁欺负了谁的分身,谁的跑(也可能两只同时在外面,那就互相点头);
- 1 小时窗口里哄过/抱过/喂过一次就既往不咎。

三态(在家 / 跑了 / 等点头)**纯从事件流派生**(`runaway`、`coax`、`forgive` 三个标记谁的 id 最新谁说了算),零新增字段。

### ✨ TA 也在线(同框)
两个人**同时开着页面**时,首页顶部亮起「**TA 正在看这只分身**」,两只镜像分身在屏幕上贴贴,还多出一个只有此刻才有的键:**🫳 摸摸头**。

- **同框期间所有互动效果 ×2**——当着面,骂更疼(委屈 +30),抱也更甜(亲密 +20)。负面一样翻倍。
- **实时看到对方在戳你**:TA 那头一发动作,你这边代表你的那只分身立刻抖一下,冒出「TA 刚戳了你 👉」。
- **切后台就自动下线**(约 25 秒内对方看到你走了),摸摸头的键随之消失。

心跳搭在 `POST /presence` 上,前端用 TanStack 的 `refetchInterval` 每 10 秒打一次——**切后台时它会自动暂停**,「在不在看」这个语义是白捡的,不需要 SSE / WebSocket。数值翻不翻倍**由服务端自己判定**,不信客户端。

> 「TA 也在」这个信号本身,就足够让人不舍得关页面。

### 🎭 剧情副本(第 4 个页签)
AI 抛一个场景——「**你俩被困在电梯里**」——然后每一幕,**两个人各选一个选项**。

- **同回合双人抉择**:谁都看不到对方选了啥,**直到两个人都选完**。齐了就同时揭晓,AI 把**两个人的选择**一起写进后果里,再推进到下一幕。
- 4 幕带选项 + 1 幕结局。**一天最多开一章**;没打完的**顺延到第二天**,不作废。
- 打完在聊天时间线上留一条纪念条。做选择也算**今日露面**(能续火苗)。
- 轮到你选时,🎭 页签亮红点;你先选完,对方会收到一条「就等你了」的推送。
- **没配 DeepSeek Key 也能完整打通**——走本地剧本库(电梯 / 末班车 / 厨房事故 / 雨天没带伞 / 深夜便利店),每一幕开头都会回显你俩刚才各自选了什么。

> 异步、慢节奏、必须两个人一起才走得下去。这是第一个「你不来,我就走不动」的玩法。

### 💬 统一聊天时间线
事件和聊天合成**一条时间线**:
- 普通动作显示成一行 tip(如「TA 戳了你」),聊天/AI 回应显示成气泡。
- **AI 气泡**和**真人(本尊)气泡**样式区分;说话人按性别上色(男蓝 / 女粉),左右对齐。
- **向上滑动懒加载**:首屏只拉最新一小页,往上滑再补更早的,不会一次性全量加载;新消息实时在底部追加。

### 🎨 移动端 & 性别主题
为手机而做的扁平 UI:`100dvh` 满屏、底部安全区适配、固定底栏和输入框、宠物"呼吸光环"随心情变色。整体主色跟随登录者性别——**男生蓝、女生淡粉**。

### 🛟 稳的那些细节
- **幂等**:连点或断网重试,同一动作对方只会收到一条,数值不会重复涨。
- **断网卖萌**:请求失败时是一句卖萌台词 + 数字不动,网络恢复自动补拉,不会白屏。
- **记住我 / 自动登录**:勾一下下次自动进,刷新不掉登录。

### 🔥 情侣火苗 + 每日一问
- **火苗**:两人**每天都露面**(任一互动或答题)才续上,连续天数越攒越高;一方完成、另一方没动时进入"就差你了"状态。漏正好一天可**花亲密续火**救回来。
- **每日一问**:每天一道 AI 混味题(暧昧 / 深度 / 沙雕轮换),**双方都答完才解锁对方答案**——答完在等 TA,就是每天回来的理由。解锁后落进聊天时间线可回看。

### 📲 消息推送(PWA + Web Push)
- 关着页面也能收到:**TA 撩你 / 本尊回你 / 每日一问对方先答了 / 剧情等你抉择 / 委屈爆表 / 分身跑了**即时推;**火苗快灭 / 已断可救 / 昨夜梦话**定时提醒。
- 「⚙️ 我」页签一键开启;离线安全:服务端没配 VAPID 密钥就整套静默关闭,不影响主功能(**梦话和出走照常发生**,只是不推送)。
- **iPhone 要先「添加到主屏幕」**(iOS 16.4+)才收得到;需 HTTPS(localhost 除外)。

## 技术栈

- **前端**:Vite + React 18 + TypeScript,TanStack Query(轮询/缓存),React Router,framer-motion(动效);Vitest + MSW 测试。
- **后端**:FastAPI + SQLAlchemy 2.0,MySQL 8(`utf8mb4`),JWT 鉴权;AI 走 DeepSeek(httpx)。
- **部署**:Docker Compose 三件套 `db` + `backend` + `web`(nginx 托管前端 + 同源反代 `/api`)。

## 快速开始

```bash
cp .env.example .env      # 可选:改端口 / 密码 / 填 DeepSeek Key
./start.sh -d             # 打包前端 + 后台起三个容器
```

浏览器打开 **http://localhost**(默认 80 端口)。

- 手机 / 局域网访问、绑定 `.local` 本地域名、当服务器长期运行等,见 **[`deploy/README.md`](deploy/README.md)**。
- 想点亮真 AI:在 `.env` 填 `DEEPSEEK_API_KEY=sk-...` 再 `docker compose up -d backend`;留空则全程走离线沙雕兜底。
- 想开消息推送:`cd backend && ./.venv/bin/python -m app.gen_vapid` 生成一对密钥,把打印的 `VAPID_PUBLIC_KEY` / `VAPID_PRIVATE_KEY` 填进 `.env`,`docker compose up -d backend` 生效。**注意 Web Push 需 HTTPS**(localhost 除外),生产要给 nginx 挂证书。

> 两个人才玩得出隔空互动的爽点——用两台设备(或两个隐身窗口)各注册一个号走一遍,对照 [`docs/couple-playtest-checklist.md`](docs/couple-playtest-checklist.md)。
